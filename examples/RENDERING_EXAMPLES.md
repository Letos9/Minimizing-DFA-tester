# Примеры режимов визуализации

Файл `automata_rendering_modes.csv` содержит примеры для проверки ограничения
в 15 состояний:

1. `n=12`, `m=36` — обычный подробный граф с подписями переходов.
2. `n=15`, `m=45` — верхняя граница: граф ещё отображается полностью.
3. `n=16`, `m=32` — первый размер, который заменяется безопасной сводкой.
4. `n=80`, `m=240` — проверка сводки на большом автомате.
5. `n=150`, `m=300` — проверка сводки на очень большом автомате.

Таким образом, в файле есть не только «удобные» примеры, но и пары по обе
стороны порога 15 состояний. Это позволяет сразу заметить ошибку вида `<`
вместо `<=`.

Повторная генерация:

```powershell
.venv\Scripts\python examples\generate_rendering_examples.py
```
